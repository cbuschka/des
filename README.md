Docker Event Scripts
===============================

version number: 0.0.1
author: Cole Brumley

Overview
--------

Run scripts in response to Docker events.

Installation / Usage
--------------------

To install use pip:

    $ pip install des


Or clone the repo:

    $ git clone https://github.com/colebrumley/des.git
    $ python setup.py install
